# 🧠 Machine Learning: Supervised Learning Paradigms

This directory contains the implementations of my Machine Learning practical labs. The primary focus is on understanding the core mathematics behind supervised learning algorithms and optimizing cost functions from scratch.

## 📂 Contents & Labs Overview

This folder includes Jupyter Notebooks detailing the following concepts:

* **Linear Regression & Optimization:**
  * Implementation of Linear Regression to predict continuous values.
  * Mathematical demonstration and application of **Gradient Descent** to minimize the cost function.

* **Classification Models:**
  * **Logistic Regression:** Implementation of binary classification models.
  * **Softmax Regression:** Advanced implementation for multi-class classification problems.

## 🛠️ Technical Stack
* **Language:** Python
* **Environment:** Jupyter Notebook (`.ipynb`)
* **Focus:** Supervised Learning, Mathematical Optimization, Cost Functions

## 🚀 How to View
You can view the code, mathematical formulas, and data visualizations directly on GitHub. Simply click on any of the `.ipynb` files in this directory to render the notebook in your browser.
