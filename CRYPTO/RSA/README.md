# 🔐 Cryptography & Defensive Security

This directory contains scripts and projects focused on secure communications, cryptographic algorithms, and defensive network infrastructure.

## 📂 Contents Overview

* **RSA Cryptography Suite (Bash):**
  * A collection of custom shell scripts implementing RSA cryptography mechanics.
  * **Key Features:**
    * Key pair generation (`generation.sh`)
    * File encryption and decryption (`encrypt.sh`, `decrypt.sh`)
    * Digital signature creation and verification (`sign.sh`, `checksign.sh`)
    * Interactive command-line interface (`menu.sh`, `display.sh`)

* **Defensive Infrastructure & Secure Communications:**
  * **IPsec VPNs & Firewalls:** Architectures for robust site-to-site IPsec tunnels and implementation of strict IP filtering rules.
  * **Layer 2 Security:** Configurations demonstrating DHCP Snooping, Port Security, and integration with Intrusion Detection Systems (IDS).

## 🛠️ Technical Stack
* **Language:** Bash / Shell Scripting
* **Cryptography:** Asymmetric Algorithms (RSA), Digital Signatures
* **Network Defense:** IPsec, Firewalls, IDS
* **L2 Protocols:** Port Security, DHCP Snooping

## 🚀 How to View
Explore the source code of the `.sh` scripts directly in this folder to review the cryptographic logic and shell automation techniques.
